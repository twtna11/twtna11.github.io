什麼是視差滾動？

視差滾動 (Parallax Scrolling) 是一種電腦圖學技術，讓背景圖像的移動速度比前景圖像慢，從而在二維場景中產生深度和沈浸感的錯覺。這種技術最初用於早期的電動遊戲，現在則廣泛應用於現代網頁設計中，以提升使用者的互動體驗。

在這個範例中，當您滾動滑鼠時，請注意觀察：

  • 月亮會快速下降 (好像在落下一樣)
  • 文字會往旁邊或下方移動
  • 後方的山脈移動緩慢
  • 前方的山脈幾乎不動或遮擋視線

設計特點

純程式碼繪製

本頁面不使用外部圖片連結，所有的山脈皆使用 SVG Path 繪製，月亮與星星則使用 CSS，確保載入速度快且不會連結失效。

JavaScript 控制

監聽 `window.scrollY` 事件，根據滾動距離動態計算每個圖層的 `transform` 位移量，實現不同速度的移動。

響應式設計

使用 Flexbox 與相對單位 (rem, vh, %)，確保在手機與電腦上都能呈現良好的視覺效果。
