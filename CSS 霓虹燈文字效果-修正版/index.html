霓虹燈文字：色彩與物理原理

觀念一：光的物理性 (Core Brightness)

為什麼文字本體是白色的？在現實世界中，霓虹燈管中心的氣體發光亮度最高，相機拍攝時會呈現 過曝 (Overexposure) 的白色。

程式碼對應： color: #fff; 和前兩層陰影 0 0 5px #fff 。

觀念二：光暈擴散 (Light Diffusion)

光線穿過空氣會散射。離光源越遠，光線越弱、邊緣越模糊，但顏色飽和度會顯現出來。

程式碼對應： 使用多層次 text-shadow 堆疊。內層銳利明亮，外層 (如 0 0 80px ) 模糊且帶有顏色。

觀念三：對比度 (Contrast)

發光效果必須建立在黑暗背景上。若背景全白，光暈將無法顯現。

程式碼對應： body { background-color: #111; }
