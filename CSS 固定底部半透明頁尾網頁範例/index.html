網頁主內容

此區塊為網頁的主要內容。您可以上下捲動頁面來測試頁尾的固定效果。

示範長內容

這是一段很長的文字，目的是確保頁面可以捲動，這樣才能測試固定頁尾的效果。固定頁尾使用了 position: fixed; 屬性，並設定 bottom: 0; 和 width: 100%; ，使其在捲動時仍然保持在螢幕底部。

半透明效果是透過 CSS 的 background-color: rgba(R, G, B, A); 語法實現的，其中 A（Alpha）值控制透明度（0.0 為完全透明，1.0 為完全不透明）。在這個範例中，我們使用了 rgba(51, 51, 51, 0.85) ，這是一個略微透明的深灰色。

為了防止內容被固定頁尾遮擋，我們在 .content 區塊設定了一個與頁尾高度相同的底部內邊距（ padding-bottom: 60px; ）。

內容不斷增加，持續延伸，直到網頁產生捲軸，讓您能清楚觀察頁尾始終貼齊螢幕底部的行為。

請記得將頁尾的 src="./placeholder.png" 替換為您的實際圖片檔案路徑。

聯絡資訊文字和圖片都透過 Flexbox 在頁尾中實現了良好的置中對齊。

這是長內容的底部... 再往下捲動也看不到頁尾消失。
