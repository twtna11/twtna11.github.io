Section 01

奇數區塊：左側進場

這是網頁的第一個區塊。當頁面載入且此區塊超過 50% 可見時，它會從 左邊 滑入。
這裡使用了 CSS Transform 與 Intersection Observer API。

Section 02

偶數區塊：右側進場

請向下捲動。當您看到這個橘色區塊的一半時，整個卡片會從 右邊 滑入。
這種左右交錯的動畫設計可以引導使用者的視線成「Z」字型閱讀。

Section 03

奇數區塊：再次左側

這是一個綠色的區塊。遵循奇數規則，它同樣是從 左側 進場。
設定 threshold: 0.5 確保使用者已經準備好閱讀內容時，動畫才開始播放。

Section 04

偶數區塊：再次右側

粉紅色的區域代表第四個章節。它從 右側 滑入。
您可以嘗試快速上下滑動，觀察動畫觸發的時機點。

Section 05

奇數區塊：最終章

這是最後一個區塊。從 左側 淡出顯示。
透過簡單的 class 切換 ( hidden-left / hidden-right )，我們可以輕鬆管理大量區塊的動畫方向。
